Navigate & Display

cd DIR [-]
Move to a directory
The current dir is your "working directory"
The path to DIR can be absolute or relative
Use - to go back
pwd
Print the working dir
ls DIR
Show contents a directory
Leave DIR blank to list the working dir
cat FILE
Print out contents of FILE
head [-n] [-c] NUM FILE
Print out the first NUM lines or bytes of FILE
-n NUM of lines
-c NUM bytes
tail [-n] [-c] NUM FILE
Print out the last NUM lines or bytes of FILE
-n NUM of lines
-c NUM bytes
open FILE
Open the file, simple enough
wc FILE
Show word and character count of FILE

Dir/File Creation

mkdir NAME
Create an empty directory within your working dir
touch NAME
Create a file (remember to include the extension)

Move, Copy, Delete

mv [-r] FILE DESTINATION
Move FILE to a new dir
Use the same dir in FILE and DESTINATION to rename
-r copy recursively (for dirs)

Examples:
mv ~/file.txt ~/dir/file.txt move
mv ~/file.txt ~/file2.txt rename
cp [-r]FILE DESTINATION
Copy FILE to a new directory
-r copy recursively (for dirs)
rm [-rfv] FILE
Delete FILE. This bypasses the trash
-r delete recursively (for dirs)
-f force deletion
-v print files as they are deleted

Combining Commands

COMMAND > FILE
Redirect output of COMMAND into FILE
Use >> to append isntead of overwriting
Example: echo "hello" > file2.txt
COMMAND_1 | COMMAND_2
Pipe the output of COMMAND_1 into COMMAND_2
Basically run COMMAND_2 with COMMAND_1 as an input
Example: head -n 5 file.txt | tail -n 2

Nice stuff

man COMMAND
Show manual for a specified command
This is handy for parameters and stuff
(ie: ls, touch, rm)
alias ALIAS="COMMAND"
Give an alias to a command
ALIAS is the new name, COMMAND may have parameters
I recommend opening ~/.zshrc and appending your alias

Examples:
alias py=python3 no params
alias ls="ls -ACFlh" include params
history
Print out previous commands
sudo powermetrics
Show metrics such as CPU temp, as well as processes
You must have admin privileges and enter your password

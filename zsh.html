File Management

cd DIR
Move to a directory
The current dir is your "working directory"
The path to said dir can be absolute or relative
mkdir NAME
Create an empty directory within your working dir
touch FILE
Create a file (remember to include the extension)
> FILE
Create and edit a file
mv FILE DESTINATION
Move a file
rm [-rf] FILE
Delete a file. This bypasses the trash
Use the -r parameter to delete a dir and all its contents
Use the -f parameter to force

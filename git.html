Repo Initialization

git init
Create an empty local repository in your working directory
git clone URL
Clone a repository from the url onto your machine

Staging

git status
View repo status and show any modified/uploaded files
git add FILENAME
Add file(s) to the repo and update what will be committed
git reset FILENAME
Unstage files while keeping changes
git restore FILENAME
Discard changes of the file(s) in your current repository.
This undoes any changes to said file
git mv FILE DESTINATION
Move a file and stage the change
git rm FILENAME
Delete the file from the repo and stage the removal
git diff FILENAME
Show diff of what is changed but not staged
git diff --staged
Show diff of what is staged but not comitted
git commit -m "MESSAGE"
Commit all staged files with a message

Branch & Merge

git branch
List all branches.
A * will appear next to the current one
git branch NAME
Create a new branch at the current commit
git checkout
Switch to a different branch
git merge BRANCH
Merge the specified branch's history into the current one

Inspect & Compare

git log
Show commit history for current branch
git log BRANCH1..BRANCH2
Show commits in branch 2 that are not in branch 1
git diff BRANCH1..BRANCH2
Show diff in branch 2 that are is in branch 1

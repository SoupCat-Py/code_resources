CTK message boxes
CTK input validation:
Code Description
%d Type of action (0 = deletion, 1 = insertion, -1 = other)
%i Index of cursor position where change is happening
%P Value of the entry after the change is done
%s Value of the entry before the change happens
%S String that is being inserted/deleted
%v Current validation mode ('key' 'focusin' 'focusout' 'forced')
%V Validation mode that triggered the callback
(same as %v but evaluated inside the function)
%w Full widget name (ie: .!ctkentry2.!entry)

Compiling

python3 -m pyinstaller
--onedir --onefile --clean --windowed --strip
--add-data "SOURCE:DEST" --icon-ICON_PATH PY_PATH
Build a .py file into an app
This works with .exe (windows) and .app (macOS)
NOTE: --windowed tells the terminal that this is a GUI app
codesign --deep --force --sign
"CERT" APP_PATH
Re-sign a .app
Before running this, you need to make a certificate using Keychain Access
You'd want to do this after editing info.plist
